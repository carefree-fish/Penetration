# SQL 注入手工分析步骤

- 判断是否存在注入，注入是字符型还是数字型
(1). 数字型: SQL语句一般不需要闭合 大多数不需要注释符号
(2). 字符型: SQL语句需要闭合 或者需要注释符号

- 猜解SQL查询语句中的字段数及字段顺序
(1). order by: 对结果集按照一个列或者多个列进行排序。
(2). UNION SELECT: 联合查询。

- 获取当前数据库名
(1). database()函数
(2). user()函数

- 获取数据库中的表名
(1). group_concat(table_name)函数
(2). information_schema表

- 获取表中的字段名

- 下载数据
